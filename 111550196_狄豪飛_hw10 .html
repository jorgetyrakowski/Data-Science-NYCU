Objective: Prepare the dataset for input into the neural network.¶

Actions:¶

  • Normalize Images: Scale the image pixel values to a range of 0 to 1 for more effective training. Since pixel values range from 0 to 255, dividing by 255 achieves this normalization.

  • Reshape Images (if using CNNs): If you're using a Convolutional Neural Network (CNN), you'll need to reshape the images to include a channel dimension. Since Fashion-MNIST images are grayscale, the channel dimension is 1.

In [7]:
train_images = train_images / 255.0
test_images = test_images / 255.0
train_images = train_images.reshape((-1, 28, 28, 1))
test_images = test_images.reshape((-1, 28, 28, 1))

Design Neural Network Architecture¶

Create a model suitable for classifying Fashion-MNIST images.

Actions:

  • Choose Model Type: For Fashion-MNIST, a CNN is generally effective.

  • Define the Architecture:

In [8]:
from tensorflow.keras import layers, models

model = models.Sequential([
    layers.Conv2D(32, (3, 3), activation='relu', input_shape=(28, 28, 1)),
    layers.MaxPooling2D((2, 2)),
    layers.Conv2D(64, (3, 3), activation='relu'),
    layers.MaxPooling2D(2, 2),
    layers.Flatten(),
    layers.Dense(64, activation='relu'),
    layers.Dense(10, activation='softmax')
])

Compile the Model¶

Prepare the model for training by setting up the optimizer, loss function, and metrics.

  • Actions:

  • Choose an Optimizer: Common choices include Adam, SGD, etc. Adam is often preferred for its effectiveness in variousscenarios.

  • Set the Loss Function: For a classification problem like Fashion-MNIST, 'sparse_categorical_crossentropy' is suitable when labels are integers.
  • Select Metrics for Evaluation: Accuracy is a common metric for classification problems.
  • Compile the Model: Bring together the optimizer, loss function, and metrics.
In [9]:
optimizer = 'adam'
loss = 'sparse_categorical_crossentropy'
metrics = ['accuracy']
model.compile(optimizer=optimizer, loss=loss, metrics=metrics)

Conclusions

Cluster Formation: There is some degree of cluster formation which suggests that the model is able to learn distinguishable features for different classes. This is a good sign that the neural network can potentially separate the classes when trained properly.

Class Overlaps: There are areas where different colors (representing different classes) seem to be overlapping or intermingled. This overlap indicates that certain fashion items share similar features and are harder to distinguish. For example, if dark blue and light blue points are overlapping, it may indicate that those two particular classes of fashion items are similar to each other.

Separability: Some colors form distinct clusters without much intrusion from other classes, indicating good separability for those classes. A neural network should be able to classify these with higher accuracy.

Dense Regions: There are dense regions where multiple classes are present. These "borderline" areas might be challenging for the model and could benefit from more complex features or models that can capture more subtle differences.
